Searching XBDD

The feature page search

Before you can search navigate to the build page. Please read Viewing Builds.

The search feature searches though all scenarios and features to find occurrences of the search words.

Features displayed larger than scenarios and scenarios have a margin to there left.

Search Fields

Search Logic

Matches are found if a scenario or feature contains any of the search words in the specified fields. Scenarios are only displayed if they contain one of the search words however features are displayed if they contain one of the search words or if any of there scenarios contain any of the search words. The results are then ordered based on the feature that has the most occurrences in its own content and the content of each of its scenarios.

When a feature is selected the feature is opened and the search box will disappear. When a scenario is selected its parent feature is opened, the scenario is scrolled to and opened. For the search box to disappear click anywhere else on the page. To bring it back click on the input search box at the top right of the page in the header.

Searching for a product

This search filters the products that are displayed. It searches for occurrences of the text string in the product names.
