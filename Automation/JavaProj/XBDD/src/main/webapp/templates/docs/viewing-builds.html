Viewing Builds

From the Product Page

There are two ways to access builds in XBDD, you can select a build from a list of all builds for a product version, or you can jump straight to the latest build of a product.

Build Selection

In order to select a build in XBDD you must first choose a product. On the landing page a list of products will be generated on the left hand side as shown.

You can scroll to find the Product or you can search for it using the Search Box.

Once you have selected a product by clicking on it the build navigation will appear from the right hand side.

You can then select the version you want with the `Version` drop down. Once you have chosen the version you want you can then choose a build from the list or search for a build number in the search box.

Latest Build

To view a products latest build click on the cog icon next to the product you want.

A drop down will appear with an option for "Latest Build", clicking this link will take you there.

From the Build Page

Once you are already viewing a build you can change which build you are looking at two ways

Build Selection

You can access the same build selection menu that you could on the main page by clicking on the build number in the menu bar

Through a Feature

You can also switch to a different build by Viewing a Feature and selecting a build from the Execution History

From the Merge View Page

You can access the same build selection menu that you could access on the main page by clicking on the build select button in the menu bar.
