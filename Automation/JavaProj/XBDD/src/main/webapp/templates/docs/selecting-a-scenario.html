Selecting a scenario

Preparatory steps

In order to edit a scenario you must first locate the build and feature that this scenario presides in.

  • For help on how to access a build please read the help for selecting a build
  • For help on how to access a feature please read the help for selecting a feature

Feature Overview

Once a feature is selected the feature overview as can be seen below will become visible in the right hand column.

In the top left corner the previous and current status can be seen, as well as when and by whom this feature was last edited.

For a more detailed view of the revision history click the link "view all revisions". More information on this can be found in the section on revision history.

In the right hand column the feature name is displayed at the top, as well as the description for that feature. Below this the complete execution history for the current version of the feature is visible, showing the status of this feature for all builds.

Selecting a scenario

Below the feature overview is a list of all the scenarios that belong to this feature. Clicking on one of these scenarios brings up information about the scenario. The current status of the scenario is colour coded into the heading of each scenario section as can be seen above.
