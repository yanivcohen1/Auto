Manual Testing

Preparatory steps

In order to edit a scenario you must first locate the build and feature that this scenario presides in.

  • For help on how to access a build please read the help for selecting a build
  • For help on how to access a feature please read the help for selecting a feature
  • For help on how to access a scenario please read the help for selecting a scenario

View scenario testing information

Once a scenario is selected its scenario test information will be visible, for more information about what these fields mean please read editing a scenario

Changing the status of a scenario

For more information about what the statuses of a scenario mean please read status of scenarios and features

The status of individual steps in a scenario can be set by clicking on the step and the colour will change to reflect to what status you are changing the step too. The ordering of change is green red and white (for no change), this order is then repeated.

In the bottom right hand corner of each scenario there are two buttons as show above, these buttons allow you to change the status of the scenario. They will cause the status of each of the steps of that senario to be changed to this status.

Save your changes

After changing the status of a scenario or status you must remember to save your changes. See the last section on editing a scenario for more details.
