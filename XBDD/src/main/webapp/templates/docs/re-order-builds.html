Getting to the re-ordering build page

To get to the re-ordering page open the build navigation for the product and version that you wish to re-order. At the bottom of the build navigation list is a button called "Reorder Builds", clicking on this button will take you to the re-ordering page.

Using the Re-Ordering page

Once on the re-ordering page you can reposition builds in a few ways, keep in mind that the most recent builds should go at the TOP of the list.

Drag and Drop

You can simply drag any build to a new position in the list

Move to Top / Move to Bottom

By hovering over a build two buttons, Move to Top and Move to Bottom will appear. Clicking either button will perform the respective function, so clicking Move to Top will move the build to the top of the list

Auto Sort

At the top of the page there is a button called "Auto Sort", this button will attempt to sort the builds automatically in the order below.

Saving Changes

Quite simply click the green "Save" button and your changes will be saved to xbdd
